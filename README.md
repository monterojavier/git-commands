# Git Commands

`git status` -> tells you all the changes and if they've been tracked or not

`git remote` -> tells you all the connections to other repos

`git clone <url>` -> makes a copy of a repo into your computer

`git checkout -b <branch-name>` -> creates a new branch

`git add .` or `git add <file-name>` -> tells git to track all your changes

`git commit -m "<message>"` -> adds a message to all the added changes

`git push <remote> <branch>` -> pushes all the changes to github

`git fetch <remote> <branch>:<branch>` -> updates your local repo to have the same branches as the remote

`git pull <remote> <branch>` -> updates your local repo to have the same code as the remote

`git merge <branch>` -> merges the changes from the branch selected into the current branch
